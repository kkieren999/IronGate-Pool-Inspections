1. Upload this site folder to your hosting.
2. Open Firebase Console.
3. Go to Authentication > Sign-in method.
4. Enable Email/Password.
5. Go to Authentication > Users.
6. Click Add user.
7. Add user:
   Email: irongate.pool.bne@gmail.com
   Password: choose a strong password
8. Go to Firestore Database > Rules.
9. Copy/paste the rules provided in ChatGPT.
10. Open:
   admin-availability.html

HOW IT WORKS
- Public booking page reads the availability collection.
- Only the signed-in admin email can create/update/delete availability dates.
- A deleted availability override returns that date to the default rule:
  weekdays available, weekends unavailable, past dates unavailable.

DOCUMENT FORMAT
Collection: availability
Document ID: YYYY-MM-DD
Example: 2026-07-01

Fields saved by admin console:
- status: available / unavailable / fully_booked
- available: true / false
- maxBookings: number
- bookedCount: number
- note: string
- reason: string
- updatedAt: timestamp
- updatedBy: email
